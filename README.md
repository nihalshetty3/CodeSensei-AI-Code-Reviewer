# CodeSensei-AI-Code-Reviewer
Imagine you wrote some code and you want a senior developer to look at it and tell you what's wrong. Normally you'd have to wait for a colleague, post on Stack Overflow, or just hope your code is fine. CodeSensei is an AI that acts like that senior developer — available 24/7, never annoyed, always thorough.

You paste your code (or connect your GitHub). The AI doesn't just stare at it and say "looks good." Instead it actually goes and does things:

It searches a database of CVEs to check if you're using a package with a known security hole
It runs ESLint on your code automatically
It looks at your team's past code to understand your conventions
It remembers that last week you made the same mistake with passwords — and calls it out again
Then it gives you back a full annotated review: exactly which line is wrong, why it's wrong, how to fix it, and a learning resource explaining the concept. Like having a mentor who has read all of Stack Overflow and your entire codebase simultaneously.

The "agentic" part means it doesn't just do one thing — it plans what to check, decides which tools to use, calls those tools, reads the results, and synthesizes everything into a coherent review. It thinks in steps, not in one shot.
